# Portfolio v2 ui

Portfolio website hosted using Cloudfront

## Setup

Install [`pnpm`](https://pnpm.io/installation)
and run: 
```bash
pnpm install
```

To run locally:
```bash
pnpm run start
```

To build ui:
```bash
pnpm run build:frontend
```

To build kill switch lambda:
```bash
pnpm run build:lambda
```
